- Add Repo
	- `flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo`
- Find App
	- `flatpak search gimp`
- Install App
	- `flatpak install flathub org.gimp.GIMP`
	- `flatpak install flathub org.mozilla.firefox`
- Rus App
	- `flatpak run org.mozilla.firefox`
- App List
	- `flatpak list`
- Update App
	- `flatpak update`
